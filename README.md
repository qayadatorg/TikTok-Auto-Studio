# TikTok Auto Studio

**Version 1.0.0** - Professional Bulk Video Uploader for TikTok

A GUI-based bulk video uploader for TikTok designed for production houses and bulk uploaders. Upload hundreds of videos with natural user behavior simulation — no API usage, just browser automation that mimics real user interactions.

## ✨ Features

- **🎯 Drag & Drop Upload** — Add videos individually or in bulk
- **⚡ Smart Queue System** — Automatic scheduling with natural delays between uploads
- **🤖 Human Behavior Simulation** — Random mouse movements, typing patterns, and delays
- **👤 Single Account Focus** — Optimized for one official TikTok account
- **📝 Bulk Metadata Editing** — Set titles, hashtags, and privacy for all videos at once
- **📊 Upload History & Stats** — Track your upload activity and success rates
- **🛡️ Anti-Detection** — Stealth browser patches to avoid automation detection
- **🎨 Beautiful Dark UI** — TikTok-themed interface built with React + Tailwind
- **🔄 Auto Update System** — Automatic update checking and download via GitHub releases

## 🎯 How to Use

1. **🔐 Login** — Click "Login to TikTok" in the sidebar. A browser window opens for manual login.
2. **📁 Add Videos** — Drag & drop videos or browse files on the Upload tab.
3. **✏️ Set Metadata** — Edit titles, hashtags, and privacy per-video or in bulk.
4. **▶️ Start Queue** — Switch to Queue tab and click "Start Queue".
5. **📈 Monitor** — Watch real-time progress. The system waits naturally between uploads.

## ⚙️ Settings

- **⏱️ Min/Max Delay**: Time gap between uploads (default: 1-5 minutes)
- **🔢 Max Daily Uploads**: Safety limit per day (default: 50)
- **👁️ Default Privacy**: Public, Friends, or Private
- **#️⃣ Default Hashtags**: Auto-applied to all new videos
- **⏸️ Pause on Error**: Stop queue if an upload fails
- **🔄 Auto Update**: Automatic download and install updates
- **🔍 Check Updates on Startup**: Check for new updates when app starts

## 🔄 Update System

TikTok Auto Studio includes an automatic update system:

- **Automatic Checking**: Checks for updates on startup (if enabled)
- **Manual Check**: "Check for Updates" button in Settings
- **User Control**: Toggle auto-update and startup checking
- **GitHub Integration**: Uses GitHub Releases API for update detection
- **Browser Download**: Opens download page in default browser

## 🔒 Security & Privacy

- **Local Processing**: All automation happens locally on your machine
- **No Data Collection**: No telemetry or data collection
- **Single Account**: Designed for personal/business account use
- **Browser Isolation**: Uses separate browser profile for automation

## 🐛 Troubleshooting

### Common Issues

1. **Login Issues**: Use "Reset Profile" if login fails
2. **Upload Failures**: Check TikTok account status and video format
3. **Queue Stuck**: Pause and restart queue if needed
4. **Update Issues**: Manual download available from GitHub releases
